# Decidim::L10n

Localization improvements for the United States.

## Usage

L10n will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-l10n"
```

And then execute:

```bash
bundle
```
