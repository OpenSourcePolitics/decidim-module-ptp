# Decidim::Ptp-budgets

Improvements to the budgets module based on the customer requirements.

## Usage

Ptp-budgets will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-ptp-budgets"
```

And then execute:

```bash
bundle
```
